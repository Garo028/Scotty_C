```bash

```

Open `http://localhost:3000`, enter your number, scan the code.

## 📋 Commands

Run `.help` or `.menu` in WhatsApp to see all 110+ commands.

### Highlights
- `.vv` — Reveal view-once media → sent to **your DM only**
- `.ai` / `.gpt` — AI chat
- `.sticker` — Convert image/video to sticker
- `.tiktok` / `.instagram` — Download media
- `.tagall` — Tag everyone in group
- `.rizz` / `.pickup` / `.hack` — Fun new commands
